Characters to avoid

  1. < is written by: &+lt+;
  2. > is written by: &+gt+;
  3. & is written by: &+amp+;
    4. *without the + sign

  • <img src = "imageName.jpg"

  • It is generally better to save the image that you add as users could experience unstable jumps while using ones from the internet. Also, one must specify width and height of the image to maintain visual layout and spacing of the image.
